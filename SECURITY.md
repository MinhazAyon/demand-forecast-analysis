# Security Policy

## Supported Versions
We actively maintain and support the latest release of this project. Older versions may not receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0     | Fully supported |

## Reporting a Vulnerability
We take security issues seriously and appreciate your help in improving the safety of this project.

If you discover a security vulnerability, please **do not open a public issue**.  
Instead, contact us directly:

- **Email**: [minhaz182219@gmail.com]

We will:
1. Acknowledge receipt of your report within 48 hours.  
2. Provide a status update and timeline for resolution.  
3. Credit you in the release notes (if you would like).  

## Responsible Disclosure
Please avoid publicly disclosing the vulnerability until we have released a fix. Coordinated disclosure helps protect the community.  

---

Thank you for helping us keep this project secure!
